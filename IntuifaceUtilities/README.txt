///////////////////////////////////////////////////////////
////// Installation Instruction ///////////////////////////
///////////////////////////////////////////////////////////

For a complete install of all DLL files and necessary folders,
follow the following steps.

1. Run 'install.bat' by double clicking it.

2. Check all the opened command terminals for any errors in 
installation.

3. Finished.


///////////////////////////////////////////////////////////
/////// DLL Installation Instructions /////////////////////
///////////////////////////////////////////////////////////

In order to update just the DLL files for a project follow the 
following steps.

1. Make sure that 'install.bat' has already been run previously
on this machine. (You should be accessing this README from
 C:\IntuifaceUtilities)

2. Replace the current DLLs in C:\IntuifaceUtilities\DLL\...
 with the updated DLLs.

3. Run 'InstallDLLs.bat' by double clicking it.

4. Check all the opened command terminals for any errors in 
installation.

5. Finished.
